HOW TO RUN

1. Open Terminal.

2. Navigate to the backend folder:

   cd backend

3. Install required packages:

   npm install

4. Start the server:

   node server.js

5. Open a browser and visit:

   http://localhost:3000

   or just run the index.html file

HOW TO USE

1. Enter a website URL (e.g., google.com, github.com, netflix.com).
2. Click "Scan Website".
3. Wait for the analysis to complete.
4. View the:

   * Security Score
   * Risk Level
   * Detected Security Issues
   * Recommendations
   * Safer Alternative Websites
